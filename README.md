# Trading Bot Baseline

Dies ist die lokale Basisstruktur für den Binance Trading Bot (Paper/Live-Modus).
Erstellt automatisch via PowerShell-Skript.

## Quickstart
1. GitHub-Repo anlegen (privat).
2. Im PowerShell-Fenster:
   git init
   git add .
   git commit -m "initial trading bot baseline"
   git branch -M main
   git remote add origin https://github.com/<DEINNAME>/trading-bot.git
   git push -u origin main

3. Danach in Codex oder CI mit Docker Compose deployen.
